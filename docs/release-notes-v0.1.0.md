# Digital Pet v0.1.0

The first public release brings Shiro, a customizable golden retriever digital
pet, to macOS and the web.

## Included

- Native always-on-top macOS app
- Framework-neutral `<digital-pet>` Web Component
- Customizable pet name with Shiro as the default
- Sit, walk, run, jump, down, sleep, roll over, handshake, hi-five, salute,
  namaste, speak, quiet, and fetch commands
- Corner naps, autonomous behavior, cursor interaction, and draggable placement
- Four display sizes
- Multiple balls and fetch behavior
- Live GitHub Pages demo and direct browser-script download
- Package archive for HTML, React, Vue, Angular, Svelte, and other web projects
